# 실습파일 제출 요령

1. topmentor@hanmail.net으로 이메일 제출 
2. 이메일 제목은 '[이름 반] x일차 실습'으로 지정 <br>
   ex) [홍길동 A] 2일차 실습
3. 실습파일 이름은 임의로 지정하여 첨부파일로 첨부하실 것 
   (1개 파일로 압축해서 보내길 권함)
